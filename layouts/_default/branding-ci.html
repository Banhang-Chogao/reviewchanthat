{{/* Typography */}}

Typography

Hệ thống chữ được định nghĩa qua CSS variables, fallback qua hệ thống font của trình duyệt để đảm bảo tải nhanh mà không phụ thuộc vào bên thứ ba.

:root {
  --font-sans: -apple-system, BlinkMacSystemFont, "Segoe UI",
    Roboto, "Noto Sans", system-ui, sans-serif;

  --font-mono: "SF Mono", Menlo, Monaco, Consolas,
    "Roboto Mono", "Noto Mono", monospace;
}

Font không được load từ CDN hay server bên ngoài. Trình duyệt tự chọn font phù hợp nhất với hệ điều hành của người dùng — macOS dùng San Francisco, Windows dùng Segoe UI, Linux dùng Noto Sans. Cách này giúp blog tải nhanh hơn, không bị phụ thuộc vào bên thứ ba, và chữ luôn sắc nét trên mọi nền tảng.

Level Spec Usage
H1 clamp(1.5rem, 4vw, 2.2rem) · ExtraBold 800 · letter-spacing: -.03em Hero title (trang chủ)
H2 clamp(1.4rem, 3vw, 1.8rem) · ExtraBold 800 · letter-spacing: -.02em Post title, section heading
H3 1.15rem · ExtraBold 800 Card title, block heading
Body 1.02rem · Regular 400 · line-height: 1.7 Nội dung bài viết
Nav .82rem · Bold 700 · letter-spacing: 0 Nav pill, badge, chip
Label .72rem · Bold 700 · Uppercase · letter-spacing: .03em Category badge, footnote
Small .68rem · SemiBold 600 Tag, metadata, caption
